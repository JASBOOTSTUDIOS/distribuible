# R — funciones flecha

- `r_flecha_basica.jasb`: IIFE `((x) => …)(arg)`.
- `r_flecha_interpolacion.jasb`: flecha dentro de interpolación `${…}`.
- `r_flecha_asignada.jasb`: `macro` y llamada como valor.
- `r_flecha_multilinea.jasb`: cuerpo con `{ … }` y `retornar`.
- `r_flecha_multilinea_llamadas.jasb`: macro multilínea que llama funciones globales y parámetros tipados (`texto`, `entero` implícito en `edad`).
