# S — funcion/retorna/retornar

- `s_estres_recursion_profundidad.jasb`: presiona stack de llamadas con recursion repetida y profundidad controlada.
- `s_estres_integracion_01.jasb`: estres de funciones recursivas, bucles, matemáticas y operadores ternarios.
- `s_estres_integracion_02.jasb`: estres de manejo de textos dinámicos, funciones que retornan textos interpolados.
- `s_estres_integracion_03.jasb`: estres de control de flujo en macros (romper, continuar), bucles anidados y condicionales complejos.
- `s_estres_integracion_04_manejo_errores.jasb`: `intentar` / `atrapar` / `final` con funciones que retornan valores.
- `s_estres_integracion_05_matematicas_pesadas.jasb`: bucles y funciones (`es_primo`) sin flechas.
- `s_estres_integracion_06_maquina_estados.jasb`: máquina de estados con `macro` y función flecha; casos **R/S** de flechas también en [tests/P1_12_funciones_flecha/](../../P1_12_funciones_flecha/).

Estrés dedicado a flechas (bucle): [tests/P1_12_funciones_flecha/S/NOTAS_S.md](../../P1_12_funciones_flecha/S/NOTAS_S.md).
