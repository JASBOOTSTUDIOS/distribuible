## 🎯 Flujos de Trabajo Comunes

### Flujo 1: Evaluar Clustering Existente
```
1. INICIO_RAPIDO_CLUSTERING.md → "Informe Completo"
2. Copiar código → ejecutar con tus datos
3. README_CLUSTERING.md → "Guía de Interpretación"
4. Analizar resultados
```

### Flujo 2: Elegir Número Óptimo de Clusters
```
1. INICIO_RAPIDO_CLUSTERING.md → "Método del Codo"
2. README_CLUSTERING.md → "Casos de Uso" → Ejemplo 2
3. Ejecutar con k=2 hasta k=10
4. Graficar inercia + Silhouette
5. Elegir k con mejor balance
```

### Flujo 3: Comparar Algoritmos de Clustering
```
1. README_CLUSTERING.md → "Casos de Uso" → Ejemplo 1
2. Ejecutar K-Means, DBSCAN, etc.
3. Generar informe para cada uno
4. Comparar múltiples métricas
5. Decidir basándose en consenso
```

### Flujo 4: Detectar y Corregir Problemas
```
1. ejemplo_clustering.jasb → ejecutar
2. Si score bajo → README_CLUSTERING.md → "Solución de Problemas"
3. Revisar Silhouette por muestra
4. Identificar puntos problemáticos
5. Ajustar algoritmo o limpiar datos
```

### Flujo 5: Entender la Implementación
```
1. RESUMEN_CLUSTERING.md → "Arquitectura"
2. metricas_clustering.jasb → leer código
3. test_metricas_clustering.jasb → ver casos de prueba
4. README_CLUSTERING.md → "Referencias" → papers originales
```

## 📦 Dependencias

### Archivos Requeridos
```jasboot
# Dentro de metricas_clustering.jasb:
usar todas de "../base.jasb"         # AnalizadorBase
usar todas de "../math_lib.jasb"     # raiz_cuadrada, etc.
```

### Funciones Externas Usadas
- `crear_lista()` - Stdlib de Jasboot
- `lista_agregar()`, `lista_obtener()`, `lista_tamano()` - Operaciones de lista
- `crear_mapa()`, `mapa_asignar()`, `mapa_obtener()` - Operaciones de mapa
- `raiz_cuadrada()` - De `math_lib.jasb`
- `str_desde_numero()` - Conversión a string
- `imprimir()` - Salida estándar

### Sin Dependencias Externas
- ✅ No requiere librerías de terceros
- ✅ Compatible con Jasboot VM 1.0+
- ✅ Implementación pura en Jasboot
