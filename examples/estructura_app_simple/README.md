# Estructa App Simple

Ejemplo minimo de una app web hecha con `Estructa` y `Jasboot`.

## Ejecutar

```powershell
node .\.vscode\run-jasb.cjs .\examples\estructura_app_simple\app.jasb
```

Luego abre:

- `http://127.0.0.1:18210/`
- `http://127.0.0.1:18210/api/info`
- `http://127.0.0.1:18210/salud`
